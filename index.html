Mac中安装Kubernete 

Guide

K8s由多个部分组成minikube为主master，kubectl为Rest client，kubernete依赖于容器环境，以及VM，所以我们需要先安装Docker环境以及VM
{: .lead}
<!–-break-–>

  • Homebrew
  • Docker for Mac
  • Minikube
  • virtualbox
  • Kubectl

Step 1 Homebrew

首先安装[Homebrew][https://brew.sh/]，和[Homebrew Cask][https://caskroom.github.io/]

Step 2 Docker

安装[Docker for Mac][https://docs.docker.com/docker-for-mac/install/].通过官方Dmg安装文件安装。

Step 3 VM

安装[VirtualBox for Mac][https://www.virtualbox.org/],同样可以通过`Homebrew`来安装

如果你提示Warning: Cask 'virtualbox' is already installed已经安装过，可以选择重新安装

如果提示Warning! Found the following active VirtualBox processes:表示Virtual Box正在运行, 可以kill掉相应的进程然后进行重新安装

启动minikube, 运行$ minikube api-version可以看到版本列表表示安装正确.

拉取所有所需的镜像到本地后就可以查看一下pod，如果均启动成功就Okey.
